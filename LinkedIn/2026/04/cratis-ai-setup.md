🚀 Device-agnostic development is no longer a feature request — it's a requirement.

At Cratis, we work across phones, iPads, Macs, random browsers — whatever's nearby. Our team lives in GitHub issues and we needed our Copilot setup to match that reality.

So we built a system that synchronizes all our AI configuration across every repository automatically. 🔄

Here's how it works:

✏️ **Edit anywhere** — Find something that needs fixing in your Copilot setup? Edit it in whatever repository you're working in.

🔀 **Sync on merge** — When changes hit main, a GitHub Actions workflow automatically syncs to every other repo in the organization via pull requests.

📚 **Reference hub** — We maintain a dedicated AI repo with best practices and setup guides so the whole team (and anyone else) can learn from what we've built.

The result? I can work from my Mac, switch to my iPad, grab my phone — and everywhere I go, my setup is exactly the same. No manual sync. No forgotten updates. No friction. Just work. 💪

No toil. No complexity. Just automation that gets out of your way.
